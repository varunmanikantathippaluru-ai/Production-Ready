Build a production-ready full-stack web application called "IntelliDocs AI".

IntelliDocs AI is an AI-powered Knowledge Assistant that allows users to upload documents and chat with them using natural language.

The platform should NOT contain dummy data.

Every user must be able to create an account, verify their email, log in securely, upload their own documents, and access only their own files and chat history.

Use real authentication and a real database.
